服務條款

最後更新日期：2026年6月

1. 服務說明

本工具（「發片工具」）為個人開發之社群媒體影片自動發布應用程式，協助使用者將影片內容同步發布至其本人擁有或管理之社群媒體帳號。

2. 使用範圍

本工具僅供開發者本人或經授權使用者使用，用於管理自身的社群媒體帳號內容發布，不對外提供商業服務。

3. 使用者責任

使用者須確保所發布之內容符合各社群平台之使用條款與當地法律規範，本工具不對使用者發布之內容負責。

4. 資料使用

本工具僅使用各平台 API 授權之存取權杖（Access Token）以執行發布功能，不會將使用者資料用於本服務說明以外之用途，亦不會將資料提供予第三方。

5. 服務變更

開發者保留隨時修改、暫停或終止本工具功能之權利，恕不另行通知。

6. 聯絡方式

如有任何問題，請透過開發者之社群媒體帳號聯繫。
