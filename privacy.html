隱私政策

最後更新日期：2026年6月

1. 概述

本隱私政策說明「發片工具」如何處理使用者資料。本工具為個人自用之社群媒體影片發布應用程式。

2. 資料蒐集

本工具僅在本機（使用者自己的電腦）儲存以下資料：

  • 各社群平台之 API 存取權杖（Access Token）
  • 使用者輸入之發布內容（文案、影片路徑）

上述資料皆儲存於使用者本機端的設定檔案中，不會上傳至任何外部伺服器或雲端資料庫。

3. 資料用途

所蒐集之資料僅用於呼叫各社群平台官方 API，以執行使用者主動觸發之影片發布功能。

4. 資料分享

本工具不會將任何使用者資料分享、出售或提供予第三方。所有 API 呼叫僅直接與各平台官方伺服器（如 Meta、TikTok、LINE、X）進行通訊。

5. 資料安全

使用者應妥善保管本機存放之 Access Token 等敏感資訊，避免外洩。

6. 使用者權利

使用者可隨時刪除本機儲存之設定檔案，以移除所有已儲存之資料。

7. 聯絡方式

如對本隱私政策有任何疑問，請透過開發者之社群媒體帳號聯繫。
